# Scénarios

Scénario 1 : Observer l'impact d'un grand nombre d'anomalies sur la durée de traitement

Scénario 2 : Observer l'évolution de la simulation en fonction de la dispersion hétérogène des anomalies

Scénario 3 : Tester la priorisation des anomalies par les drones avec la mesure de l'intensité

Scénario 4 : Simulation d'une catastrophe naturelle de grande envergure
